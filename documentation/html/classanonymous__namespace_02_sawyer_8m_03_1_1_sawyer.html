function anonymous_namespace{Sawyer.m}::Sawyer::SawyerVolume ( in  self)

SawyerVolume generates a point cloud of possible endefector positions. From this it calculates the working volume of the sawyer itself and returns the results.

Return values
volumethe volume of the sawyer workspace in the form : [maxX, minX, maxY, minY, maxZ, minZ, totalVolume]; MaxX - max reachable x position minX - min reachable x position MaxY - max reachable Y position minY - min reachable Y position MaxZ - max reachable z position minZ - min reachable z position totalVolume - working volume of the sawyer
